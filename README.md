# 🏪 Inventory Management and Sales Tracking System (IMSTS)

## 📌 Overview
The **Inventory Management and Sales Tracking System (IMSTS)** is a database-driven solution designed to improve how retail businesses manage stock, suppliers, and sales transactions.

This project was developed as part of the CMPG 311 Database Project and focuses on designing a structured, scalable, and reliable database system.

---

## 🎯 Objectives
The system aims to:

- Improve accuracy of inventory and sales records  
- Reduce stock shortages and overstocking  
- Enable real-time stock updates  
- Support efficient supplier management  
- Ensure data consistency and integrity  
- Provide reliable data for reporting and decision-making  

---

## ⚙️ Key Features

- 📦 Product management (cataloging, pricing, stock levels)  
- 🔄 Stock movement tracking (stock-in, stock-out, adjustments)  
- 💰 Sales transaction recording  
- 🚚 Supplier management and tracking  
- 📊 Reporting support (inventory, sales, low stock)  
- 🔐 Role-based access control and user management
